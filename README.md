## Node.js

Run `node -v`.\
If it shows a version of node, skip this step

### Installing node

Go to [Node.js](https://nodejs.org/en), install the latest version, everything as default

## When you have node installed
Run `npm i` once.\
Run `npm run dev` to serve the webapp
